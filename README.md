# lab8-pygame

Simple Pygame lab project.

## Setup (Windows PowerShell)

1. Create the virtual environment:
   ```powershell
   python -m venv .venv
   ```
2. Activate it:
   ```powershell
   . .\.venv\Scripts\Activate.ps1
   ```
3. Install dependencies:
   ```powershell
   pip install -r requirements.txt
   ```

## Run

```powershell
python main.py
```
